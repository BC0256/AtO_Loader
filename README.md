## Usage

 1. Make a folder in BepInEx\plugins called `cards`
 2. Place files into:
	BepInEx\plugins\cards\{cardName}\{cardName}.json
	BepInEx\plugins\cards\{cardName}\{cardName}.png

## JSON template
Minimum for card to show up  
```json
{
	"cardName": "SampleCard",
	"id": "SampleCard",
	"showInTome": true
}
```
